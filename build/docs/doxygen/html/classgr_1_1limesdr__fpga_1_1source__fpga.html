Return a shared_ptr to a new instance of limesdr_fpga::source_fpga.

To avoid accidental use of raw pointers, limesdr_fpga::source_fpga's constructor is in a private implementation class. limesdr_fpga::source_fpga::make is the public interface for creating new instances.

Parameters
serialDevice serial number. Cannot be left blank.
channel_modeChannel and mode selection A(1), B(2), (A+B)MIMO(3).
filenamePath to file if file switch is turned on.
